# Arold Portfolio

Modernes persönliches Portfolio, entwickelt mit **React**, **TypeScript**, **Vite** und **Tailwind CSS**.  
Dieses Projekt dient dazu, mein Profil, meine Fähigkeiten und meine Projekte zu präsentieren sowie eine direkte Kontaktmöglichkeit anzubieten.

## Überblick

Dieses Portfolio beinhaltet:

- eine moderne und responsive Startseite
- eine professionelle Vorstellung meiner Person
- eine Projektübersicht
- eine Skills- und Kompetenzsektion
- ein funktionierendes Kontaktformular
- Mehrsprachigkeit (**Deutsch, Englisch, Französisch**)
- modernes Design mit Animationen und responsivem Layout

## Verwendete Technologien

- React
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion / Motion
- i18next / react-i18next
- Lucide React
- Formspree

## Funktionen

- responsives Design für Mobile, Tablet und Desktop
- Sprachwechsel mit Länderflaggen
- flüssige Navigation zwischen den Sektionen
- Download des Lebenslaufs (CV)
- Kontaktformular mit E-Mail-Versand über Formspree
- moderne Animationen (Scroll & Hover)
- Integration von GitHub, GitLab, LinkedIn und E-Mail
